操作説明
WASDでディスプレイの左側で上下左右に移動する
幼児龍の時：左側shiftでダッシュできる。障害物に当てられたらダメージを受ける。
　　　　　　ダッシュ期間はゲーム流れがスローされ、無敵フレームをつける。MPを消費。
　　　　　　経験値が微妙に上がる
　　　　　　一定値になったら、経験値ケージが点滅しspaceで成長する
　　　　　　成長期間は無敵時間
成体：成体になった瞬間に噴火
　　　ダッシュ機能なし、障害物を破壊できる
　　　ｊでレーザーを撃つ、ｋで噴火、連打オッケー、ＭＰ注意

レベルが上がるときhp,mp,上昇＆リセット、攻撃力上昇
ゲームレベルが変換する時、敵が暴走、弾連打が一定時間で続く
成体に成長したあと、特殊エネミーが生成、特殊エネミーを倒せばスコアが上昇

残り：
UI
特殊敵のアニメーションと攻撃手段と落ちるアイテム（経験値、バリア、道具など）
ラスボス
レベルによるステージの変化
